The Knight's Tour

This project simulates the knights tour in a console application. The language that was used to create this project was C#. When the application runs, a simulation takes place which shows the number of moves it takes to "walk" the knight chess piece across the entire chessboard. The application demonstrates the use of recursion and back trancing. The end result shows a message after every 1,000,000 turns.
